O que é USABILIDADE?

Usabilidade: Medida na qual um produto pode ser usado por usuários específicos para alcançar objetivos específicos com eficácia, eficiência satisfação em um contexto específico de uso.

Teste de usabilidade do Vets Haustier

Os testes foram aplicados em cinco pessoas com idades variadas para ter uma ideia real do público que irá utilizar o produto, desde jovens até idosos, onde a grande parcela de usuários está entre idades 20 a 60 anos. O protótipo teve score 87,0 sem aplicação de recursos de acessibilidade avançados que poderiam ajudar na utilização do aplicativo. Assim fica classificado na nota A, adjetivo Excelente, aceitação Aceitável e NPS Promotor.

Comentários sobre os resultados dos testes

PRIMEIRA QUESTÃO:

A maioria dos usuários optou por usar o sistema mas não com tanta frequência.

SEGUNDA QUESTÃO:

A maioria dos usuários não achou o sistema complexo, porém um usuário achou o sistema complexo em uma escala de 3. O perfil deste usuário é uma idosa de 60 anos e informou que teve dificuldade em encontrar as informações referentes as principais doenças que se encontram na tela do pet, o problema pode ser resolvido com um atalho na tela principal.

TERCEIRA QUESTÃO:

A maioria dos usuários achou o sistema fácil de usar.

QUARTA QUESTÃO:

2 usuários não acharam que precisariam de ajuda para usar o sistema, outros dois acharam que talvez para realizar algumas tarefas precisariam de um grau de ajuda e 1 usuário informou que precisaria de ajuda para realizar as tarefas de pagamento de serviços.

QUINTA QUESTÃO:

2 usuários acharam que as funções estavam bem integradas outros 2 que estavam parcialmente integradas e 1 usuário informou que não achou as funções bem integradas e sugeriu ajustes na apresentação das informações (algo mais claro e direto sobre os serviços - o qual foi ajustado).

SEXTA QUESTÃO:

A maioria dos usuários não achou que que o sistema apresenta inconsistências e 1 usuário achou que existiam algumas inconsistências.

SÉTIMA QUESTÃO:

A maioria concordou que aprenderiam rápido utilizar o sistema.

OITAVA QUESTÃO:

Os usuários não acharam o sistema atrapalhado de usar.

NONA QUESTÃO:

3 usuários se sentiram confiantes ao usarem o sistema e 2 se sentiram parcialmente confiantes.

DÉCIMA QUESTÃO:

3 usuários informaram que não precisaram aprender nada de novo para usar o sistema, 1 usuário informou que teve que aprender algo para realizar algumas tarefas e 1 usuário informou que precisou aprender a como solicitar um serviço(escolher horário, tipo de serviço - métodos de pagamento).
