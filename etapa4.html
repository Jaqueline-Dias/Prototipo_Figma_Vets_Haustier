SEMANA 12 - Padrões de projeto

Padrões de projeto

Christopher Alexander cunhou o conceito de padrão de projeto ao observar que certas soluções, na área de arquitetura predial, eram recorrentes. A primeira definição de padrão de projeto estabelecida por ele, em 1979 (*), é a seguinte: “cada padrão é uma regra de três partes, que expressa uma relação entre um certo contexto, um problema e uma solução”. Um padrão de projeto, portanto, consiste na documentação de uma solução para um problema que ocorre em um contexto específico. Segundo Alexander, para resolver um particular problema em um dado contexto é preciso levar em conta as "forças conflitantes" em tal contexto (exemplo, quarto precisa ser ensolarado, mas não superaquecido no verão) que precisam ser consideradas e harmonizadas em um possível solução. Depois apresenta uma solução e a justificativa para tal solução. Um padrão deve ser geral o suficiente para poder ser aplicado a uma gama de situações em um particular contexto e o específico suficiente para orientar o projetista nas suas decisões. A proposta é projetar segundo as forças existentes no particular contexto

Nomeação e Logo

A escolha da logo e do nome Vets Haustier se deu pelo fato de tentar inovar e criar um nome marcante. Usei o termo Vets para remeter a idéia de veterinários e o termo Haustier para fazer referência a Pet, pois Haustier é um termo em alemão que significa Pet ou animal de estimação.

Protótipo

Após realizar todas as etapas anteriores, chegou a hora de dar forma ao app através da construção das telas e navegação.

Acessando a tela de Login

Ao acessar o aplicativo, o usuário realiza o Login ou cadastro na ferramenta inserindo os seus dados pessoais. O Login é realizado através do email do usuário.

Acessando a tela inicial / Cadastrando o Pet / Selecionando um Pet já cadastrado

Ao realizar o Login, o usuário se depara com a tela onde são visualizados os serviços veterinários ofertados. Caso o usuário não tenha nenhum Pet cadastrado, ele deve clicar na opção "Adicionar Pet" para cadastrar um novo Pet. A tela exibe um formulário para o usuário inserir os dados do seu Pet, como: Tipo, Nome, Raça, Data de nascimento e Sexo. A adição do Pet é feita por meio do botão “Cadastrar”

Escolhendo o serviço veterinário desejado

Em seguida, o usuário é direcionado para a tela de escolha dos serviços veterinários. Deslizando pelos Cards é possível ter acesso a todos os serviços ofertados e clicando em "agendar" para selecionar o desejado.

Escolhendo o serviço / Realizando agendamento

Após escolher o serviço desejado, é hora de escolher a melhor data e horário para receber o veterinário. O app exibirá um calendário que listará as datas e horários disponíveis. A partir desta tela, o usuário poderá navegar para a tela de agendamento onde é possível escolher a especialidade do serviço e para quais pets o serviço será prestado. Ao clicar no botão "Continuar" o usuário é direcionado para as telas de pagamento.

Confirmando pagamento

Após definir a data e horário do agendamento e forma de pagamento, é necessário clicar no botão "Continuar" para confirmar o pagamento.

Exibindo detalhes do agendamento

Ao confirmar o pagamento, o app exibe um resumo com todos os detalhes do tipo de agendamento contendo as informações de status de pagamento, serviço e data selecionada para atendimento. O app também será capaz de notificar o usuário sobre todos os acontecimentos referentes a seus agendamentos.
